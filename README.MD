# Vetta MVP

Vetta is an AI-powered risk assessment platform designed for Buy Here Pay Here (BHPH) car dealerships.  
It helps dealerships make smarter financing decisions by evaluating customers through behavioral and financial questions,  
then generating a risk score (Low, Medium, High).

---

## 🚀 Features
- **Dealership-specific logins** with Clerk authentication
- **AI-powered risk analysis** using OpenAI
- **Supabase database** with Row-Level Security for per-dealership data
- **Printable QR cards** auto-generated for each dealership
- **Analytics dashboard** to track scans, assessment starts, completions, and drop-offs
- **Modern UI** with dark mode support

---

## 🛠 Tech Stack
- **Frontend**: Next.js (React)
- **Backend**: Supabase Edge Functions
- **Authentication**: Clerk
- **AI Integration**: OpenAI API
- **Hosting**: Vercel
- **Database**: Supabase (PostgreSQL + RLS)
